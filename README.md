# nextjs-learning

Next.jsのキャッチアップ用リポジトリ。

## 構成

- [nextjs-playground/](./nextjs-playground/) — 学習用Next.jsプロジェクト本体。トピックごとに手を動かして学ぶ
- [forge/](https://github.com/kousukef/forge) — Claude Code用のツール設定。使い方に慣れるためにフォークして導入（.gitファイルは削除済み）

## 進め方

`nextjs-playground/app/` 配下にトピックごとのルートグループ（例: `(routing-basics)`）を作り、そこに動作確認用のページを実装しながら学習する。1プロジェクト内でトピックを分ける方針とし、パッケージのバージョン差など環境ごと変えたい特別な検証をする場合のみ別プロジェクトを作成する。

学んだ内容や環境構築のメモは [nextjs-playground/docs/notes/](./nextjs-playground/docs/notes/) にまとめる。
